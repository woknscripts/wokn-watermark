# wokn-watermark

Standalone FiveM watermark script. Each player can independently position and resize their own watermark via /watermark.

SETUP
-----
1. Drop the wokn-watermark folder into your resources directory
2. Place your watermark image as watermark.png in the root of the resource folder
3. Add `ensure wokn-watermark` to your server.cfg

USAGE
-----
- /watermark  — Opens the editor overlay for the player running the command
- Drag the watermark box to reposition it anywhere on screen
- Drag the white corner handle to resize
- Press Enter to save and close
- Press Esc to cancel without saving
